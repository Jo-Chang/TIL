## Git branch
>   독립적인 작업흐름을 만들고 관리
---
```
(master) $git branch {branch name}
```
>   브랜치 생성
---
```
(master) $git checkout(switch) {branch name}
```
>   브랜치 이동
---
```
(master) $git checkout -b {branch name}
```
>   브랜치 생성 및 이동
---
```
(master) $git branch
```
>   브랜치 목록
---
```
(master) $git brach -d {branch name}
```
>   브랜치 삭제
